# MoJian_ARPG（部分未完成）

### 游戏介绍：
主要包括背包，战斗系统，Boss副本，包括三个场景：创建角色、主场景、战斗场景

### 开发工具：
Unity2018.2.7 + NGUI

### MVC架构
Model:数据层
View:视图层
Controller：处理与数据视图以外的层

### 设计模式：
代理模式、观察者模式、单例模式
